Java edition vs Bedrock edition

There are 2 main editions to minecraft, bedrock, and java (also education edition).
although they're both Minecraft, there is some differences between them.

Supported Devices

Minecraft java only supports Windows and mac devices, while bedrock can be played on windows,
console, and smart-phones.

Modding

There are a lot of mods on Minecraft java that can completely change the games look and gameplay.
However, Minecraft bedrock's modding is limited, and mods in the bedrock marketplace is paid.

Combat

Minecraft bedrock uses a simpler combat system, where you can swing your sword as fast as you want.
In java, you have to wait for your sword to cooldown, but axes are also stronger.

UI (user interface)

Bedrock's UI is bigger so its friendly to smartphone players, while java's ui is smaller, due to
it being only available to desktop devices.

which should I choose?

If you aren't on windows or mac, or you want to play with friends on different devices,
or you want a casual experience, then pick bedrock. However, if you want good modding support,
or all your friends have a pc, or you don't mind a harder combat system, and you're on windows or mac, then play java.
